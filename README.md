# AI PROXY

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/ai-proxy.git
cd ai-proxy
go build -o ai-proxy .
```

## Usage

```bash
./ai-proxy --help
```
