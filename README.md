## Ai-proxy

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/ai-proxy.git
cd ai-proxy
go build -o ai-proxy .
```

### Usage

```bash
./ai-proxy --help
```

## Quick Start

```bash
./ai-proxy [command]
```
